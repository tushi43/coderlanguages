Machine Learning (ML)

Machine learning is the science of getting computers to act without being explicitly programmed. Machine learning is a field of computer science that often uses statistical techniques to give computers the ability to "learn" with data. For this, a huge amount of data is passed to machine to give better result by analyzing the data.
Python, Java, and R are most popular skills to learn for mastering Machine Learning.

Natura Language Processing (NLP)

Natural Language Processing (NLP) is an area of research and application that explores how computers can be used to understand and manipulate natural language text or speech to do useful things. It is widely used to check emotions of people by reading the reviews which automatically rank the product. It can be used to detect fake reviews or autogenerated article by using semantic analysis.
Applications of NLP include a number of fields of studies, such as machine translation, natural language text processing and summarization, user interfaces, multilingual and cross language information retrieval (CLIR), speech recognition, artificial intelligence and expert systems, and many more.
You can use Python and Java for NLP or softwares like OpenNLP, CoreNLP, Mahout and more.
