Surface web, Deep web and Dark web

Surface Web:

It is all the data which we can search on google or any other search engine, it includes all the websites, images and other publicly available code.

Deep Web:

Deep Web consists of all the private data specific to a person or oraganization. It includes Google drive data, data on cloud, Huge databases, files record of governments and other private data. This data is only accessible by authorized authority.

Dark Web:

It is place where illegal activities are done which we cannot search through google or any other search engine, it can be accessed using tor brower.

Don't try to access dark web, it is illegal.
