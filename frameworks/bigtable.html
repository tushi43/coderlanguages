BigTable

Bigtable is a distributed, column-oriented, compressed, high performance, and proprietary data storage system developed by Google Inc. Even if you have not heard about iy you had used it, this is Google's BigTable to store and process data. Developing Bigtable was started to fulfill need of companies to store and process big data(which is in petaBytes) easily and efficiently. This system is written using C++ , Java, Python, Go, Ruby languages. Bigtable integrates easily with popular Big Data tools like Hadoop, as well as Google Cloud Platform products like Cloud Dataflow and Dataproc. Bigtable supports the open-source, industry-standard HBase API, which makes it easy for development teams to get started. Bigtable offers low latency and high throughput at any scale or application . Bigtable is designed to handle massive workloads at consistent low latency and high throughput, so it's a great choice for both operational and analytical applications, including IoT, user analytics, and financial data analysis. Bigtable was designed to support applications requiring massive scalability. From its first iteration, the technology was intended to be used with petabytes of data.
It is used by Google which means all google applications.
