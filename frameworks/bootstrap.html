Bootstrap

Bootstrap is the popular framework build using HTML, CSS and JavaScript for front-end development only. It is popular because of his 12 grid system(learn more here), icons(favicons), inbuilt classes, Responsiveness, capable of handling all size screen with grid system. You can build a front-end website within 5 hours if you know how to use bootstrap, it's that easy. A dozens of components, JavaScript plugins, typography, form controls, and even a web-based customizer is provided by Bootstrap.
If you want to override any css of a class in Bootstrap make a separate css file overwrite the class with your desired attribute(s) and place it on bottom of the links of stylesheet, Thus changes can be done.
There are many templates of bootstrap available on the internet some are free and some you have to buy, they may include JQuery. Many Bootstrap plugins need Jquery for animations and responsiveness.
Bootstrap works good with icons(favicons and glyphicons) built in which can be used. You can get list of all the favicons, classes on w3schools.
This website homepage also uses Bootstrap for front-end User Interface. Bootstrap is free and open source, so can be downloaded or a link can be used in ‹head› tag for BootstrapCDN.
If you want to see the difference how the website vary in mobile screen and desktop screen size, view this both websites in your mobile w3schools and Hackerrank , see the difference by yourself.
