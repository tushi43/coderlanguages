Drupal

Drupal is a free and open source Content Management System (CMS) build using PHP that allows organizing, managing and publishing your content. This popular PHP framework is used for for web content management among global enterprises, governments, higher education institutions, and NGOs. Drupal provides a back-end framework for at least 2.3% of all web sites worldwide and increasing. Drupal publishes a single web site or shares content in multiple languages across many devices. It is Highly Scalable, secure,and has flexible content architecture. Drupal can support a diverse range of web projects ranging from personal weblogs to large community-driven sites.

Symfony

Symfony is a web-based content management system (CMS) that enables users to create and manage websites and web applications of all shapes and sizes—from the simplest of blogs to bustling news sites and feature-packed social networks. It is based on PHP and MySQL and uses XML and XSLT as its support. It also has the ability to focus only on data structures to create a responsive web design. Thus it can be both a content management system and a web application framework similar to Ruby on Rails or Django. Other projects like Drupal, laravel, PHPBB also uses symphony.

CakePHP

CakePHP is a one of the PHP framework, it provides Authentication facilities, Console tools, Controllers, Plugins. CakePHP makes building web applications simpler, faster, while requiring less code. A framework offering a flexible database access layer and a powerful scaffolding system that makes building both small and complex systems simpler, easier.CakePHP uses well-known software engineering concepts and software design patterns, such as convention over configuration, model–view–controller, active record, association data mapping, and front controller.

Zend Framework

Zend Framework is a open source, object oriented framework which consists of collection of professional PHP packages. It provides end-to-end encryption and focuses more on security. It uses Composer as a package dependency manager; PHPUnit to test all packages. It has extensible MVC implementation supporting layouts and PHP-based templates by default.

CodeIgniter

CodeIgniter is a popular and powerful open-source PHP framework to develop better websites with a small footprint of about 2MB, created by Rick Ellis. It follows MVC architecture and thus provides better performance than any other PHP framework. CodeIgniter is not strictly based on the MVC development pattern. Using Controller classes is a must, but Models and Views are optional. It also focuses on security of applications. CodeIgniter was made to be a simple and elegant toolkit, enabling rapid development of both web sites and web applications, attracting thousands of talented PHP developers.It’s also an ideal choice if you want to avoid PHP version conflict.
