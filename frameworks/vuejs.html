Vue

Vue (called view) is a progressive framework for building user interfaces easily where progressive means it can work for every user, regardless of his/her browser choice. It makes state management very simple and intuitive due to his reactivity which is one fo the main feature of Vue framework. Vue is also perfectly capable of powering sophisticated Single-Page Applications when used in combination with modern tooling and supporting libraries. Vue provides routing, templates and is distributed in self-contained components. Vue.js is a good choice for dynamic applications, as it has virtual DOM it offers higher performance, which is beneficial for complex apps.
