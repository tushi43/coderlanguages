CSS




Cascading Style Sheets describes how the HTML element will be displayed on the web page, its color, orientation and also the behavior can be decided by CSS.
CSS is like the soul to the body of HTML, at present all the colorful presentation and graphics you see on the web pages are due to CSS.
CSS is designed primarily to enable the separation of presentation and content, including aspects such as the layout, colors, and fonts.Hence a single css file can be used to design hundreds of html pages as it works on tags. CSS focuses more on the visual layout of web page.
CSS helps in reducing redundant code using a external css file which can be applied to many html documents. With the help of CSS, animations can be added to web pages. All little animations and color or structure change induces interactivity of web page with user.
