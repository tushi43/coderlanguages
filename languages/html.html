HTML




HyperText Markup Language is a markup language used to build static web pages. Markup Languages are used to format text.
It is the Building block for web development. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.
It uses tags to describe the content.In early days, HTML was used to link all the static webpages using hyperlinks and gave importance to internet as InterNet means International Network aims to connect/link the web pages.
HTML is not case sensitive. It is a long typed (Verbose) language and many programmers don't considered it as a language as ot is easy.
HTML is interpret by web browsers to compose text, images, and other material into visual or audible web pages.Each Page created in HTML contains the data to be included in the web page and the HTML tags. The Web Browser understands those tags and displays the correspoding Web Page. Many scripting or programming languages can be embed into HTML and used efficiently.
HTML has Document Structure which can contain Headings, Hyperlinks, paragraph, images, audio, videos, tables, Frames, Forms.HTML is incredibly successful, it is also limited in its scope, the tags it makes available do not provide any information about the content they encompass, it only has instructions on how to display that content,Hence XML became popular which has customized information.
