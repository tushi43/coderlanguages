Javascript




JavaScript is the most popular scripting language used to infuse dynamism snd interactivity in Web Pages. JavaScript is mostly confused with Java Programming language by beginners but they both are different. A number of the most in-demand frameworks are JavaScript frameworks — it’s the only language versatile enough to build frontend, backend, mobile and browser extensions (Still developing). Hence, JavaScript is ruling the web. Most often, employers want developers who know AngularJS, Node.js, and React.
Javascript is a high-level, dynamic, weakly typed, case-sensitive, multi-paradigm language.
Javascript initially was client-side language but now JavaScript engines are now embedded in many other types of host software, including server-side in web servers and databases, and in non-web programs such as word processors and PDF software, and in runtime environments that make JavaScript available for writing mobile and desktop applications, including desktop widgets.
I love JavaScript because only one data type var, anonymous functions, interactivity with HTML tags and easy array syntax and ability to add animation to web page. The capability of JavaScript is huge and vast.
JavaScript can be embed in a HTML page using ‹script› tags. A Single JavaScript file with .js extension can be used in many HTML pages by embedding one line also called an external js file. for example,
‹ script src="myjavascriptfile.js" ›‹/script›
