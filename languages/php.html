PHP




Hypertext PreProcessor(PHP) is a server-side scripting language designed for making dynamic web pages. Some featues of PHP are that it gives access control, File upload facility(file related operations are possible and easy), HTTP based authentication control, Associative arrays, Third party application support and have extensible architecture.
PHP code can be embedded into HTML code, or it can be used in combination with various web template systems, web content management systems, and web frameworks. You can develop an entire functional website just using HTML,CSS and PHP. It provides a ton of power and is the beating heart of monster sites like WordPress and Facebook. PHP is a must-learn language for aspiring web developers. You have to install work with XAMPPor Wamp Server to run and test your php script. PHP iwas developed as an alternative to ASP(Active server pages) and JSP(Java Server Pages) with new features.
